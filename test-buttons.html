Hierarchy Tree Toolbar Button Fix

Issue Description

The three toolbar buttons in the hierarchy tree modal were using var(--gray-700) which is a fixed dark color (#404040). This caused them to appear dark in both light and dark modes, making them inconsistent with the theme.

Solution

Changed the button background from var(--gray-700) to var(--card-bg), which adapts to the theme:

  • Light mode: --card-bg = #F0F0F0 (light gray)
  • Dark mode: --card-bg = #1a1a1a (dark gray)
