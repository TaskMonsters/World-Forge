## üé® Custom Fields

### How to Use
1. Open any "Add" modal (Location, Character, etc.)
2. Scroll to "Custom Fields" section
3. Click "+ Add Field"
4. Enter field name (e.g., "Climate", "Language")
5. Enter field value
6. Add more fields as needed
7. Remove fields with X button

- **Locations**: Climate, Language, Currency, Landmarks
- **Characters**: Occupation, Skills, Equipment, Fears
- **Factions**: Motto, Headquarters, Uniform, Rituals
- **Any module**: Add fields specific to your world
